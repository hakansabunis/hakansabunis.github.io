Autonomous Car Project

Project Overview

For the successful completion of the autonomous car project, me and my four teammates successfully developed and implemented a mathematical model and artificial intelligence approach using the OpenCV library. We demonstrated object detection and speed limit recognition by adding four signs to the track: Speed 5cm/sec, Speed 10cm/sec, Speed 20cm/sec, and a Stop sign.

During the demonstration, the autonomous vehicle correctly set the speed limits and adjusted its speed accordingly within 2-3 seconds. Additionally, the vehicle adhered to the track's boundaries, ensuring it stayed inside the 50 cm wide road and avoided crossing the white lines on both sides.

Key Achievements

  • Successfully implemented object detection and speed limit recognition using the OpenCV library.
  • Accurately recognized and adjusted speed based on traffic signs within 2-3 seconds of detection.
  • The vehicle stayed within the track’s boundaries, adhering to the white line restrictions on both sides.

Technologies Used

  • Library: OpenCV for image processing and object detection
  • Microcontroller: Arduino Uno
  • Programming Language: C++ for embedded systems
  • Sensors: Ultrasonic sensor for obstacle detection
  • Motor Driver: L298N Dual H-Bridge Motor Driver
  • Camera: ESP-32 Camera Module

Challenges Overcome

  • Initial camera image quality issues resolved by replacing the ESP32 camera with a mobile phone camera.
  • Encountered coding errors and cable connection issues were resolved through teamwork and testing.

Conclusion

This project demonstrates the effective use of mathematical models, artificial intelligence approaches, and the OpenCV library for autonomous driving. It was a valuable learning experience, enhancing skills in embedded systems, sensor integration, and algorithm development.
